Multimídia

É uma forma de comunicação que combina diferentes formas de conteúdo como texto, áudio imagens, animações ou vídeo em uma única apresentação.

A tag iframe é usada para incorporar outro documento ao documento HTML.

Tag vídeo

É usada para incorporar conteúdos de vídeo em um documento. O source é utilizado para especificar diferentes fontes de vídeo as quais o navegador abrirá a primeira que suportar.

Tag de áudio

é usada para incorporar áudios ao documento. O uso do source especifica diferentes fontes de áudio as quais o navegador abrirá a primeira que suportar. Existe a Biblioteca de áudio do YouTube a qual oferece conteúdos de som gratuitamente.

Tag picture

É uma tag usada para implementar um conjunto de imagens ao site. Especifica elementos com o source onde o navegador escolherá a imagem mais adequada.
