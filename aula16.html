CSS

O CSS e serve para estilizar o conteúdo HTML. Com ele é possível alterar cores, sombras, fontes, posicionamento, etc.

Há três formas de usar o CSS
Inline Os códigos CSS são definidos dentro de uma tag HTML
Afeta somente a tag em que foi definida
Incorporado Os códigos CSS são definidos no cabeçalho, dentro da tag head
Afetam somente o documento em que foi definido
Externo Os códigos CSS são definidos em um arquivo externo
Afetam todos os documentos que estão linkados ao CSS
