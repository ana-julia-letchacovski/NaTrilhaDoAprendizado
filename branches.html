Aula 12 - Branches e Hospedagem

Branches

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • novas funcionalidades;
  • correção de problemas;
  • alteração de código.
Branche Master ou Main
braço principal de um projeto em um repositório
Branch Secundária
braço secundário de um projeto em um repositório

Hospedagem

O GitHub Pages oferece hospedagem gratuita de sites.Basta criar uma conta e um repositório público e a URL será:

https://nomeusuario.github.io/repositório

Execução:

Locais de uso:

  • GitHub Desktop
  • GitHub
  • VisualStudio
  1. Fazer login no GitHub
  2. Sincronizar com GitHub Desktop
  3. Criar novo repositório público
  4. Realizar push do repositório mantendo-o público
  5. Criar/Iniciar o arquivo que servirá de branch main
  6. Salvar o arquivo e realizar commit seguido de push
  7. Criar branch secundaria
  8. Realizar push da branch secundaria
  9. Abrir arquivo na branch secundaria e realizar as alterações
  10. Salvar arquivo e realizar commit e push das alterações
  11. É possível criar mais de uma branch secundária
  12. Se houver certeza das alterações realizadas na branch secundária, e quiser sincronizar o main com tais alterações, realiza-se o merge
  13. Realizar push do merge
  14. Se houver outra branch secundaria é necessário atualiza-la com o novo arquivo main

Execução Hospedagem:

  1. Logar no GitHub
  2. Entrar no repositório que se pretende hospedar e acessar configurações/detalhes
  3. Descer a página até localizar o enunciado "Páginas GitHub" e clicar em "Confira aqui!"
  4. Localizar o enunciado "Source", selecionar a branch principal e salvar
  5. Entrar no link que aparecerá na tela e então será direcionado para a página de hospedagem gratuita
